Deformed algebras

Naïve q-deformed oscillator

There are various routes to q/Q-deformations. Although these are often referred to as “quantum-deformations”, this seems to be partly a muddling of the history. These deformations were originally studied in classical number theory and so on. I have the impression that once the ideas were applied to quantum-like systems, and the fact that the letter “q” or its uppercase version were used as the parameter, which also is the first letter of “quantum”, and voilà.

Here we will start from a deformation of the boson commutator: \([a,a^\dagger]_q=aa^\dagger-qa^\dagger a=1\). In terms of the function \(\Phi\), one gets a recurrence relation: \(\Phi(N+1)-q\Phi(N)=1\). [Some researchers use an upper-case “Q” to disguish it from the q-parameter for the slightly different q-deformed commutator, to be considered in the next section.] If we assume \(\Phi(0)=0\), so the states are bounded from below, the recurrence leads to a sum for the states: \[\Phi(n)=\sum\limits_{k=0}^{n-1}q^k=\frac{q^n-1}{q-1},\] since this is our old friend, the geometric sum. The value for \(q=1\), the normal boson/harmonic oscillator case, is a bit tricky in this format, since the expression has a pole at this point. However, expressing it as \[\Phi(n)=\frac{(1+\Delta q)^n-1}{\Delta q}\approx n\] in the limit \(\Delta q\rightarrow0\), returns the expected \(\Phi\) function.

We plot \(\Phi\) at values representative of \(q\) less and greater than 1.

When \(q\lt1\), the \(\Phi\) function is bounded above. When \(q\ge1\), there is no bound. Further, for \(q\gt1\), \(\Phi\) tends to exponential behaviour.

It may be of interest to consider negative \(q\) (difficult to graph for non-integer \(n\) as values of \(q^n\) become complex).

We sweep through \(q\) values in the range \([-1.5,1.5]\). For \(q\ge1\) the \(\Phi\) values increase without limit. For q in the open range \((-1,1)\) \(\Phi\) has values between 0 and a bound of \(1/(1-q)\). For positive \(q\) the \(\Phi\) values increase steadily to this limit. For \(q\lt0\) the progress is not monotonic. For \(q\lt-1\) the value of \(\Phi\) is not bounded above or below. And the negative values are “unphysical”, since they represent the value of \(a^\dagger a\) on \(|n\rangle\), which must be positive for the Born probability interpretation, as explained earlier. For \(q=-1\) we have another old friend, the Fermi-Dirac anticommutator. The Bose-Einstein commutator is, of course, \(q=1\). By considering \(\Phi(2)=1+q), we can see that \(q\) is required to be real in this case.
