# Central difference operator

## Start

The central difference operator $\delta f(x)=f(x+1/2)-f(x-1/2)$ has convolution-type basic series:

$$\delta q_n=q_{n-1}$$

starting from $q_0(x)=1$ and $q_n(0)=0$, $n>1$.
