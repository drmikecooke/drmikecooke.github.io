Joining the unitarians

We have seen that there is a close connection between real rotations in 2-dimensions and 1-dimensional rotations in the complex plane, represented by multiplication by unit-modulus complex numbers. Really not hugely surprizing once you get used to the idea of complex numbers, \(-1\) having a square root, and all that. However, there is a bit of me that still finds it remarkable at 60+.

The group of 2-dimensional rotations is called SO(2), while the 1-dimensional complex rotation group is U(1), the 1-dimensional unitary group. Both these groups are parameterized by one variable, \(\omega_{12}\) in our case.

If we had included the reflections, the rotation-reflection group would be O(2), the 2-dimensional orthogonal group. The complex analogue would be generated by complex conjugation (\(i\rightarrow-i\)) and U(1)-rotations, which then becomes a bit of a mess, particularly if you want to use it in the context of analytic function theory.

For real \(\omega_{12}\in\mathbb R,\) there is a mapping onto SO(2) and U(1) that respects addition in \(\mathbb R.\) This is a result of the identity \(\exp(a+b)=\exp(a)\exp(b),\) when the component objects \(a\) and \(b\) commute. This can be derived from Taylor series or products, along with limit arguments.

So we have \(\exp[(\omega_{12}+\omega'_{12})\mathrm J^{12}]=\exp[(\omega_{12}\mathrm J^{12})\exp(\omega'_{12}\mathrm J^{12}).\)

At this point we should notice that while there are group mappings (homomorphisms) from \(\omega_{12}\in\mathbb R\) onto SO(2) or U(1) (\(\omega_{12}\rightarrow \mathrm{SO(2)/U(1)}\)), the reverse is not a mapping. Each element of the target groups correspond to an infinite number of values of \(\omega_{12}.\) The equivalent values are separated by \(2\pi,\) a complete turn, since the Euler exponential relation assumes we are using radians. In particular, the identity elements of SO(2)/U(1) are reached by integral multiples of \(2\pi,\) which we can represent as \(2\pi\mathbb Z.\)

There are well known techniques for defining a quotient set where we divide out this \(2\pi\mathbb Z\) ambiguity of the \(\omega_{12}\) parameter to give a set of unique parameters for each element of SO(2)/U(1). This set is represented as \(\mathbb R/2\pi\mathbb Z.\)

The most common way in practice is to choose a set of representatives, e.g. \(\{\omega_{12}:0\le \omega_{12}\lt 2\pi\}.\) Now in this set we want addition to result in a value in this range. If the addition is less than \(2\pi,\) we are done. Otherwise the addition is adjusted to drag the value back into the desired range by adding or subtracting integral multiples of \(2\pi.\) Inverse elements are given by \(2\pi-\omega_{12}.\)

A more theoretical technique is to define a map \(\mathbb R\rightarrow\mathbb R/2\pi\mathbb Z\) where the elements of the target set are sets of all elements that map to a particular rotation/complex number. So if one parameter \(\omega_{12}\) leads to a rotation/complex number, the corresponding element of \(\mathbb R/2\pi\mathbb Z\) is given by \(\omega_{12}+2\pi\mathbb Z=\{\omega_{12}+2\pi k:k\in\mathbb Z\}.\) Another notation for this equivalence class is \([\omega_{12}],\) and \(\omega_{12}\) is referred to as a representative of the class. In this context, addition is defined by addition of classes: \(\{a\}+\{b\}=\{a+b\}.\) Some checks are needed to show that \([\omega_{12}]+[\omega'_{12}]=[\omega_{12}+\omega'_{12}],\) but one finds that this is indeed the case.

The practical definition is just achieved through choosing a unique representative element of each equivalence class. Indeed for some applications, a more balanced range is used such as \(\{\omega_{12}:-\pi\lt \omega_{12}\le \pi\}.\) Because we want to use the continuity of the groups, generally connected and continuous ranges are used, although this is not strictly necessary at this stage.

In abstract group theory, these observations are extended to mappings between general groups, resulting in a swathe of homomorphism and isomorphism theorems. In our case we note that our SO(2)/U(1) groups are effectively the same (isomorphic) as \(\mathbb R/2\pi\mathbb Z\) with the defined addition. We express this as \(\mathrm {SO(2)}\cong\mathbb R/2\pi\mathbb Z\cong\mathrm {U(1)}.\) This is an equivalence relation, which also gives isomorphy of SO(2) and U(1).
